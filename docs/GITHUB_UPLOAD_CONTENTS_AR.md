# محتويات الرفع إلى GitHub بعد مرحلة الطاقة

ارفع النسخة الموجودة في حزمة `github_ready` كما هي. تتضمن:

- الكود المصدر والاختبارات.
- الإعدادات والبروتوكولات المجمدة.
- بيانات تطوان المعالجة كل ساعتين، دون الملف الخام.
- نتائج حالة الطاقة الكاملة وقرارات ما قبل الاختبار.
- بصمات التجميد وسجل المحاولات المتوقفة والـerratum الوصفي.
- تقارير Development والتناثر البنيوي وحالة الطاقة.
- مقاطع LaTeX والجداول الجاهزة للمخطوط.
- الرسوم الناتجة عن حالة الطاقة.

لا تتضمن الحزمة:

- الملف الخام المنزّل من المرآة.
- بيئات Python وملفات cache.
- مخرجات LaTeX المؤقتة.
- ملفات ZIP أو RAR القديمة.
- جداول المرشحين الاصطناعية الكبيرة غير الضرورية للعرض على GitHub.

قبل إرسال الورقة، يجب تنزيل ملف تطوان من UCI مباشرة وتشغيل أداة التحقق المذكورة في `docs/TETOUAN_DATASET_MANIFEST.json`.
